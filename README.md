# Agriculture and Economic Booms

CAVEAT: This project is still a work in progress. All code is highly tentative.

This repository hosts code (but not most data, because a significant portion of the data is not supposed to be publically distributed) for an economics project about the effects of economic booms on agricultural production in the United States, focusing on the late 20th century. All code is in R.

There are six code files that are available here right now:
- cbpdata.R cleans County Business Patterns data
- agcensus.R cleans Agricultural Census data (from the ICPSR version)
- combineddata.R combines the two data sources and cleans some supplementary datasets
- summarystats.R generates summary statistics
- regressions.R runs most of the regressions
- shiftshare.R runs the shift-share instrument regressions
